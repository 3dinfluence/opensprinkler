This is a GUI-based Firmware Update tool for OpenSprinkler.

1. You need a regular size A Male to B Male USB cable (most)
   printers use this type of cable).
2. Linux users: run this program in sudo mode.
3. Windows users: you need to install
   - USBtinyISP driver (OpenSprinkler 1st-gen and 2.0s)
   - USBasp driver OpenSprinkler 2.1 and above)
4. Install the avrdude program
   - Linux: sudo apt-get install avrdude
   - Mac: AvrMacPack or OSX-AVR
   - Windows: WinAVR

Go to www.opensprinkler.com->Firmware Update for additional
details about driver installation and avrdude.
